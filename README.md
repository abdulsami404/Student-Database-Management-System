# 🎓 Student Management System API

A backend API for managing student records using FastAPI, MySQL, and SQLAlchemy.

## 🚀 Features

- Create new students
- Retrieve student by ID
- Update student details
- Delete student
- Data validation using Pydantic
- MySQL database integration

## 🛠️ Tech Stack

- FastAPI
- MySQL
- SQLAlchemy
- Pydantic
- Uvicorn
